DVD: A YouTube-DL Client For Android

Feb 22, 2021
1 min read
100DaysToOffload Tags

Video downloader app powered by youtube-dl.

For downloading YouTube videos, I usually rely upon Newpipe but, a few days back, I came across an application on F-Droid called DVD that uses youtube-dl.

youtube-dl is a command-line program to download videos from YouTube.com and a few more sites

I tried using the application to download videos off from YouTube for offline viewing purposes. Being powered by, youtube-dl I can download videos not just from Youtube but from 1000+ sites supported by youtube-dl. The application also allows specifying a custom folder to keep the downloaded video/audio files. It provides an option to update the youtube-dl version from settings and run custom commands.

The only issue that I faced throughout my usage was the slow downloading speed compared to using youtube-dl on PC. Overall, DVD is a good application for having a portable version of youtube-dl on the go.
