Inspired by this post, I also wanted to add a comment section to my blog. But the first hurdle was how to get it working and which service to use. Instead of banging my head against the wall, I looked up at Hugo’s documentation on comments. Over there, I found Utterances, an open-source, GitHub comments widget built on GitHub Issues.

Utterances specify how to set up the comment section in a very straightforward manner. But there was a problem once I did the initial set up and then went back to my blog to notice that the widget was not appearing. To fix this problem, I went to their GitHub issues and found someone having the same issue. After the following solution that was listed over there, the comment section widget started to appear.
